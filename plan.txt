ke Supabase. Artinya:

SOLUSI YANG KITA PILIH → SUPABASE SEBAGAI JEMBATAN:

Supabase sudah terkonfigurasi di supabaseClient.js ✅

DATABASE SUPABASE - TABEL YANG AKAN DIBUAT
====================================================================

[1] store_profile (sinkron dari Android)
    Kolom:
    - id: int (PK, default 1 — hanya 1 row)
    - name: text
    - address: text
    - phone: text
    - qris_image_url: text (URL gambar QR yang diupload admin)
    - updated_at: timestamptz

[2] products (sinkron dari Android)
    Kolom:
    - id: text (PK)
    - name: text
    - price: int
    - stock: int
    - created_at: bigint
    Catatan: Android push produk ke sini setiap ada perubahan.

[3] services (diinput dari Android)
    Kolom:
    - id: uuid (PK, default gen_random_uuid())
    - name: text
    - price: int
    - description: text
    - is_active: boolean (default true)
    - created_at: timestamptz (default now())

[4] customers (diisi dari web saat registrasi)
    Kolom:
    - id: uuid (PK, default gen_random_uuid())
    - nama: text
    - no_telepon: text (UNIQUE — sebagai identifier login)
    - jenis_motor: text  → 'matic' | 'gigi' | 'kopling'
    - merk_motor: text
    - plat_nomor: text
    - created_at: timestamptz (default now())

[5] web_orders (order dari web customer)
    Kolom:
    - id: uuid (PK, default gen_random_uuid())
    - customer_id: uuid (FK → customers.id)
    - customer_name: text (denormalized untuk kemudahan Android)
    - customer_phone: text
    - order_type: text → 'service' | 'product' | 'mixed'
    - items: jsonb → [{id, name, price, qty, type}]
    - total: int
    - payment_method: text → 'cash' | 'qris'
    - status: text → 'pending' | 'confirmed' | 'done' | 'cancelled'
    - is_read_by_admin: boolean (default false) ← untuk notif Android
    - created_at: timestamptz (default now())

ALUR HALAMAN LENGKAP
====================================================================

ALUR 1: PERTAMA KALI (Belum punya akun)
  /register → Form Daftar
    → Isi: Nama, No HP, Jenis Motor, Merk Motor, Plat
    → Klik [DAFTAR] → Insert ke Supabase table: customers
    → Simpan session ke localStorage
    → Redirect ke /home

ALUR 2: SUDAH PUNYA AKUN (Kembali lagi)
  / → Cek localStorage → kalau ada session → /home
  / → Kalau tidak ada → pilihan [Daftar] atau [Login]
  /login → Masukkan No HP → Cari di customers
         → Kalau ada → simpan ke localStorage → /home
         → Kalau tidak → tampilkan "No HP tidak terdaftar, daftar dulu"

ALUR 3: DI HALAMAN UTAMA (/home)
  → Tampilkan: "Selamat Datang, {nama}! 👋"
  → Info motor: {merk} {jenis} - {plat}
  → Nama toko dari store_profile Supabase
  → 2 card menu utama:
      [🔧 Jenis Servis]  →  /services
      [🛒 Beli Produk]   →  /products

ALUR 4: SERVIS (/services)
  → Ambil dari Supabase: services (is_active = true)
  → Tampilkan card tiap servis: nama, harga, deskripsi
  → Tombol [+ Tambah] → masuk ke cart state
  → Floating button [Lihat Keranjang (n)] → /cart

ALUR 5: PRODUK (/products)
  → Ambil dari Supabase: products (stock > 0)
  → Tampilkan kartu produk: nama, harga, stok
  → Tombol [+ Tambah] → masuk ke cart state
  → Search / filter produk (opsional)
  → Floating button [Lihat Keranjang (n)] → /cart

ALUR 6: KERANJANG (/cart)
  → Tampilkan semua item (servis + produk jadi satu)
  → Bisa ubah qty atau hapus item
  → Summary total
  → Tombol [LANJUT BAYAR] → /payment

ALUR 7: PEMBAYARAN (/payment)
  → Tampilkan ringkasan order
  → Pilih metode:
      [CASH] → Tampilkan: "Silakan bayar ke kasir"
               → Tombol [Konfirmasi Order]
               → Insert ke web_orders (status: pending)
               → Tampilkan halaman sukses "Order dikirim ke kasir!"

      [QRIS] → Ambil qris_image_url dari store_settings
             → Tampilkan modal dengan:
                 - Gambar QR merchant
                 - Total yang harus dibayar (tulisan besar)
                 - Instruksi: "Scan QR ini, masukan nominal Rp{total}"
             → Tombol [✅ Sudah Bayar]
               → Insert ke web_orders (status: pending)
               → Tampilkan halaman sukses
